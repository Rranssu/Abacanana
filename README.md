# Abacanana  
### Smart Environmental Monitoring for Abaca and Banana Production

Abacanana is a mobile-based environmental monitoring application developed to assist in monitoring **temperature** and **humidity** conditions for abaca and banana-related agricultural environments.

The application was created using **Expo React Native** and is designed to provide a simple, accessible, and efficient monitoring solution for local agricultural use cases.

---

## Overview

The system allows users to monitor environmental data in real time through a clean and user-friendly mobile interface.

Abacanana aims to support smarter agricultural monitoring by helping maintain suitable environmental conditions for abaca and banana production.

---

## Features

- 🌡️ Real-time temperature monitoring
- 💧 Real-time humidity monitoring
- 📱 Mobile-friendly interface
- ⚡ Lightweight and responsive design
- 📊 Simple data visualization
- 🌿 Designed for agricultural monitoring applications

---

## Technologies Used

- ⚛️ React Native
- 🚀 Expo
- 🟨 TypeScript
- 📡 Sensor-based monitoring integration

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/abacanana.git
```

### Navigate to the project directory

```bash
cd abacanana
```

### Install dependencies

```bash
npm install
```

### Start the Expo development server

```bash
npx expo start
```

---

## Building APK with EAS

```bash
eas build -p android --profile preview
```
